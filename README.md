# Claude Camp W2 练习

## 练习1：学员花名册管理器
- **功能：** 添加、查询、删除、修改学员信息，数据保存到本地 JSON 文件
- **运行：** `python student_manager.PY`
- **技术点：** 字典、JSON 读写、异常处理、命令行交互

## 练习2：文本词频统计器
- **功能：** 输入一段英文，统计每个单词和字母出现次数，按频率排序输出
- **运行：** `python counter.py`
- **技术点：** 字典计数、字符串处理、sorted 排序

## 练习3：待办事项清单
- **功能：** 添加待办（含截止日期）、标记完成、查看清单（按日期排序），数据保存到本地 JSON 文件
- **运行：** `python todolist.PY`
- **技术点：** 列表、字典、JSON 读写、异常处理、排序

## 练习4：安全计算器
- **功能：** 支持复杂算式、连续计算、清零、异常处理
- **运行：** `python calculator.py`
- **技术点：** eval、while True、try/except、字符串处理
